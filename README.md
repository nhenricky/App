# 📝 Todo List em Tkinter

Um aplicativo simples de lista de tarefas (Todo List) desenvolvido em Python utilizando a biblioteca Tkinter para a interface gráfica.

## 📌 Funcionalidades

- Adicionar tarefas
- Remover tarefas selecionadas
- Visualizar todas as tarefas
- Interface gráfica simples e intuitiva

## 🛠️ Tecnologias

- Python 3
- Tkinter

## 🚀 Como executar

1. Clone este repositório:

```bash
git clone https://github.com/nhenricky/App.git
```

2. Acesse a pasta do projeto:

```bash
cd App
```

3. Execute o aplicativo:

```bash
python main.py
```
